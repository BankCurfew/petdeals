## วิธีใช้ (ทำทุกสินค้า)

1. เลือก **3-5 keywords** จากตารางหมวดที่เกี่ยว
2. ใส่ **primary keyword ใน H1** + paragraph แรก
3. ใช้ **long-tail เป็น H2** (format คำถาม)
4. ใส่ **comparison keyword เป็น H2** + ตาราง HTML
5. ใส่ **problem keyword เป็น FAQ**
6. ใช้ **สแลง** ตลอด: ทาสแมว, เจ้านาย, น้องเหมียว
7. ใส่ "อัพเดทล่าสุด: DD/MM/YYYY"
